Rock, Paper, Scissors Game

Description:
This is a simple Python-based Rock, Paper, Scissors game that allows you to play against the computer. 
You can choose between 'r' for Rock, 'p' for Paper, and 's' for Scissors.

How to Run:
1. Clone the GitHub repository to your local machine:
   git clone https://github.com/LuaiMags/DevOpsAssignment4.git

2. Run Locally:
   Navigate to the directory and execute the Python script:
   python magh0019_Assignment4.py

3. Run via Docker:
   - Pull the Docker image from Docker Hub:
     docker pull luaimaghur/rock-paper-scissors:latest

   - Run the container:
     docker run -it luaimaghur/rock-paper-scissors
